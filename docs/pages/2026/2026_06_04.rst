 Results
............

 - Implemented a synthetic data generator with abTEM (Bloch waves).
   The script creates a system with random orientation and does a 
   rotational scan. With resources on Scarf, we can generate 10-15 
   datasets per day of a similar quality to the paracetamol data we
   initially trained our GBT model on.

 - Created a CBF exporter for simulated abTEM data. The synthetic data
   can now be imported into DIALS, and we can index the spots and process
   them in the same way as an experimentally measured dataset (Fig. 1).
 - Corrected the first draft of the AutoED paper with David W.
